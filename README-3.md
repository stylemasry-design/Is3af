# 🏥 المنيا الصحية — دليل طبي شامل

**دليل طبي رقمي شامل لمحافظة المنيا، مصر**
تطبيق ويب تقدمي (PWA) يعمل على الهاتف والمتصفح — مع إمكانية التحويل لـ APK.

---

## 📱 مميزات التطبيق

| الميزة | التفاصيل |
|--------|----------|
| 🔍 بحث ذكي | ابحث بالاسم، التخصص، القرية، المركز |
| 🗂️ تصنيفات | أطباء · مستشفيات · صيدليات · معامل · وحدات صحية · طوارئ |
| 📍 قريب مني | تحديد الموقع وعرض أقرب منشأة |
| 🗺️ خريطة | عرض المنشآت على خريطة تفاعلية |
| 👤 بروفايل كامل | صفحة تفاصيل لكل منشأة مع صور وتقييمات |
| 🌙 داكن / فاتح | دعم الوضع الليلي |
| ❤️ مفضلة | حفظ المنشآت المفضلة محلياً |
| 📴 بدون إنترنت | يعمل offline بعد أول تحميل (PWA) |
| 🚨 طوارئ | أرقام الطوارئ الرسمية مباشرة |

## 🚀 طريقة النشر

### 1. نشر مباشر على المتصفح (أبسط طريقة)
```
افتح index.html مباشرة في المتصفح
```

### 2. نشر على GitHub Pages (مجاني)
```bash
# 1. أنشئ repo جديد على github.com
# 2. ارفع كل الملفات
# 3. Settings → Pages → Deploy from main branch
# الرابط: https://username.github.io/repo-name
```

### 3. نشر على Netlify (مجاني + سريع)
```
1. اذهب لـ netlify.com
2. اسحب مجلد المشروع كله على الصفحة
3. سيعطيك رابط فوراً مثل: https://minya-health.netlify.app
```

### 4. نشر على Vercel
```bash
npm i -g vercel
cd minya-health-pwa
vercel --prod
```

### 5. تحويل لـ APK (للهاتف)
**راجع ملف APK-GUIDE.html للتفاصيل الكاملة**

## 📲 تثبيت كـ PWA على الهاتف

### Android (Chrome)
1. افتح الرابط في Chrome
2. ستظهر رسالة "إضافة للشاشة الرئيسية"
3. اضغط "تثبيت" أو "Add to Home Screen"

### iPhone (Safari)
1. افتح الرابط في Safari
2. اضغط زر المشاركة ↑
3. اختر "أضف إلى الشاشة الرئيسية"

## ⚙️ إضافة بيانات جديدة

لإضافة طبيب/مستشفى جديد، افتح `index.html` وأضف في مصفوفة `DATA`:

```javascript
{
  id: 'dr_new_001',         // معرف فريد
  type: 'doctor',           // doctor | hospital | pharmacy | lab | health_unit | emergency
  plan: 'free',             // free | premium | platinum
  verified: false,          // هل موثق؟
  name: 'د. الاسم الكامل',
  specialty: 'التخصص',
  phone: '01XXXXXXXXX',
  whatsapp: '01XXXXXXXXX',  // أو null
  address: 'العنوان التفصيلي',
  center: 'minya',          // راجع قائمة المراكز أدناه
  village: 'اسم القرية',
  hours: '09:00-21:00',     // أو '00:00-23:59' للمفتوح 24 ساعة
  rating: 4.5,              // أو null
  reviews: 50,              // أو null
  tags: ['تأمين صحي', 'حجز مسبق'],
  lat: 28.0871,             // خط العرض (اختياري)
  lng: 30.7618              // خط الطول (اختياري)
}
```

## 🔧 التخصيص

### تغيير الألوان الرئيسية
في `index.html`، ابحث عن `:root` وعدّل:
```css
--teal: #0a7c6e;      /* اللون الرئيسي */
--gold: #c9960a;      /* لون الذهبي */
```

### إضافة إشعارات Push
في `sw.js`، قسم `Push Notifications` جاهز للربط بـ Firebase FCM.

### ربط بقاعدة بيانات
يمكن استبدال مصفوفة `DATA` الثابتة بـ:
- **Firebase Firestore** (مجاني للمشاريع الصغيرة)
- **Supabase** (مفتوح المصدر)
- **Google Sheets API** (بدون كود backend)
